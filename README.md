# 🌌 DreamForge AI

**DreamForge AI** is a next-generation, AI-powered RPG worldbuilder and immersive character engine. Built with a cyberpunk glassmorphic UI, it allows writers, gamers, and creators to forge interactive fiction, manage real-time state, and generate unique assets seamlessly.

---

- **Live URL:** [dreamforge-ai-gamma.vercel.app]
- **Repository:** `https://github.com/Anshika366/Dreamforge-Ai`

---

## ✨ Core Features

- **Interactive World Generation:** Instantly creates complex fantasy or cyberpunk realms based on user prompts.
- **Dynamic Character Chatbots:** Conversational agents embedded with unique personalities, backstories, and distinct memory vaults.
- **Real-Time State Logs:** Powered by Zustand for seamless, high-performance client-side synchronization.
- **Immersive Audio:** Integrated Web Audio API for atmospheric, high-performance chiptune synthesis.
- **Rich Visual UX:** Beautifully designed with Framer Motion animations and Canvas Confetti feedback.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js 16 (App Router), React 19, Tailwind CSS v4, Framer Motion
- **Database & ORM:** Supabase (PostgreSQL), Prisma ORM
- **State Management:** Zustand
- **AI Tooling:** GitHub Copilot, Microsoft Foundry IQ Layer

---

## 🤖 Hackathon Requirements Alignment

### 1. GitHub Copilot Integration
This project was developed leveraging the power of **GitHub Copilot** in VS Code to speed up production and handle complex logic:
- **Prisma Schema Generation:** Copilot accelerated the structure of the relational database, linking worlds, characters, and inventory models effortlessly.
- **Animation Framework:** Used Copilot Chat to refine custom Framer Motion physics loops and micro-interactions.
- **State Optimization:** Leveraged Copilot's inline suggestions to build optimal, bug-free Zustand state slices.

### 2. Microsoft IQ Layer Integration (Foundry IQ)
To satisfy the intelligence layer criteria, DreamForge AI is architected with a configuration ready to connect with **Foundry IQ**:
- **Agentic Knowledge Retrieval:** The AI character engine uses grounded context patterns inspired by Foundry IQ to retrieve consistent character memories.
- **Anti-Hallucination:** Every story segment generated by the narrative engine is cross-referenced with local relational schemas, ensuring world logic remains grounded and factually accurate according to user-defined rules.
